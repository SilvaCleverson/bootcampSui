# Smart Contracts

Os smart contracts Move do CronoCapilar estão localizados na pasta raiz do projeto:

```
../sources/desafios/cronocapilar.move
```

## Módulo Principal

- **Módulo:** `cronocapilar::profile`
- **Package ID (Mainnet):** `0x313d461e1eb560b5215d9ac61f02e7f2413ddfca3d000df76ea3fa39ab680663`

## Estruturas

- `Profile` - Perfil de cuidados capilares
- `Treatment` - Registro de tratamentos
- `Event` - Eventos capilares (Big Chop, Cortes, etc.)

## Funções Principais

- `create_profile` - Criar perfil on-chain
- `register_treatment` - Registrar tratamento
- `register_event` - Registrar evento capilar
