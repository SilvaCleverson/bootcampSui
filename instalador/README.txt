INSTRUÇÕES RÁPIDAS — INSTALADOR DE 1 CLIQUE (SEM EMOJIS)

1) Extraia o ZIP.
2) Clique em **Instalar_Sui.bat**.
3) Confirme a janela do Windows (UAC) e aguarde.
4) Ao final, abra um PowerShell novo e teste:
   - sui --version
   - git --version
   - code --version

O que será instalado: Sui CLI, Git, Visual Studio Code (via Chocolatey).
Se algo falhar, consulte o arquivo **instalacao_sui.log** na mesma pasta.
